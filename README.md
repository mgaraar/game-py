# game-py 

A personal Python playground — games, utilities, and a lot of learning along the way.

This repo is where I practice writing clean code, build things I actually enjoy, and get comfortable with real development workflows like Git version control.

---

## Inspiration

This project was inspired by [Mosh Hamedani](https://github.com/mosh-hamedani), whose teaching style made me realize that good code isn't just about making things work — it's about building the right habits from the start.

## Getting Started

Want to try one of the projects locally?

```bash
# Clone the repo
git clone https://github.com/mgaraar/game-py.git

# Move into the project folder
cd game-py

# Install dependencies (if any)
pip install -r requirements.txt

# Run a project
python project_name/main.py
```

> Each project lives in its own folder. Check the folder for any project-specific notes.
